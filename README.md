# Python Millionaire Game

A simple quiz-based game inspired by "Who Wants to Be a Millionaire".

## What this does
The program asks multiple-choice questions. Each correct answer increases the prize amount.
If the user gives a wrong answer, the game ends.

## Features
- Multiple questions
- Prize system
- Game ends on a wrong answer

## Why I made this
I built this while practicing Python concepts like lists, loops, and conditionals.
